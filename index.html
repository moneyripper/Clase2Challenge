1. <html>...</html> El elemento raíz

Todas las páginas web comienzan con el elemento html. También se llama elemento raíz porque está en la raíz del árbol de elementos que componen una página web. Para crear el elemento html, escribe una etiqueta de apertura <html> seguida de una etiqueta de cierre </html>. Todo lo demás en su página web va entre estas 2 etiquetas: 


    <html>
        (Todos los otros elementos)
    </html>

2. <head>...</head> El encabezado del documento

El elemento head contiene información sobre la página web, a diferencia del contenido de la página web en sí. Hay muchos elementos que puedes poner dentro del elemento head, como por ejemplo:

  • title,título de la pestaña.
  • link, que puede usar para agregar hojas de estilo y favicons a su página.
  • meta, para especificar elementos como conjuntos de caracteres, descripciones de páginas y palabras clave para motores de búsqueda.
  • script, para agregar código JavaScript a la página.

Aquí hay un elemento head típico :


    <head>
        <title>10 Tags HTML más utilizados</title>
        <meta charset="UTF-8">
        <meta name="description" content="10 tags mas usados en el lenguaje HTML">
        <meta name="keywords" content="html,web,paginas">
        <link rel="stylesheet" type="text/css" href="/style.css">
        <link rel="shortcut icon" href="/favicon.ico">
    </head>

3. <title>… </title> El título de la página

El elemento title contiene el título de la página. El título se muestra en la pestaña del navegador (la barra en la parte superior de la ventana del navegador), así como en los marcadores, los resultados del motor de búsqueda y muchos otros lugares.

El título debe describir el contenido de la página de manera sucinta y precisa. Intente darle a cada página de su sitio un título único.

He aquí un ejemplo:

  
    <title>10 Tags HTML más utilizados</title>

4. <body>...</body> El contenido de la página

El elemento body aparece después del elemento head en la página. Debe contener todo el contenido de su página web: texto, imágenes, etc. Todas las páginas web tienen un solo elemento body, con la excepción de las páginas de conjuntos de marcos, que contienen elementos frame en su lugar.

Aquí está el formato general del elemento body:

    <body>
        (Todo el contenido de la pagina va aquí)
    </body>

5. <h1>...</h1> Un título de sección

Los encabezados le permiten dividir el contenido de su página en partes legibles. Funcionan de forma muy similar a los títulos y subtítulos de un libro o un informe.

HTML en realidad es compatible con 6 elementos epígrafe: h1, h2, h3, h4, h5, y h6. h1es para los encabezados más importantes, h2es para los subtítulos menos importantes y así sucesivamente. Normalmente no será necesario utilizar más de h1, h2y h3, a menos que su página sea muy larga y compleja.

A continuación, se muestra un ejemplo de un h1 elemento de título:

    
    <h1>10 tags HTML más utilizados</h1>

6. <p>...</p> Un párrafo

El elemento p le permite crear párrafos de texto. La mayoría de los navegadores muestran párrafos con un espacio vertical entre cada párrafo, dividiendo muy bien el texto.

Si bien puede crear "párrafos" de texto simplemente usando <br> etiquetas para insertar líneas en blanco entre fragmentos de texto, es mejor usar p elementos en su lugar. No solo es más ordenado, sino que les da a los navegadores, motores de búsqueda y otros no humanos una mejor idea de cómo está estructurada su página.

Aquí hay un ejemplo de un párrafo:

   
    <p>Esto es un texto de ejemplo</p>

Una buena regla general al escribir texto para la web es asegurarse de que cada párrafo contenga un solo punto, tema o pensamiento. Si quieres hablar de 2 cosas diferentes, usa 2 párrafos.

7. <a>...</a> Un enlace

Uno de los elementos más importantes de una página web, el elemento a le permite crear enlaces a otro contenido. El contenido puede estar en su propio sitio o en otro sitio.

Para crear un enlace, envuelve con <a> </a> etiquetas alrededor del contenido que deseas usar para el enlace y proporciona la URL a la que deseas enlazar en el atributo de la <a>etiqueta href.

A continuación, se explica cómo crear un texto que se vincule a www.example.com:


    <a href="http://www.example.com/">Visita esta pagina web de ejemplo!</a>
Visita esta pagina web de ejemplo!

8.<img> Una imagen

El elemento img le permite insertar imágenes en sus páginas web. Para insertar una imagen, primero cargue la imagen en su servidor web, luego use una <img>etiqueta para hacer referencia al nombre de archivo de la imagen cargada.

He aquí un ejemplo:

    
    <img src="myphoto.jpg" alt="My Photo">

El alt atributo es obligatorio para todas las img etiquetas. Es utilizado por navegadores que no muestran imágenes para brindar texto alternativo al visitante.

9. <div>...</div> Un contenedor de contenido a nivel de bloque

El elemento div es un contenedor genérico que puede usar para agregar más estructura al contenido de su página. Por ejemplo, puede agrupar varios párrafos o encabezados que tengan un propósito similar dentro de un elemento div .

Normalmente, los elementos div se utilizan para cosas como:

  • Encabezados y pies de página.
  • Columnas de contenido y barras laterales.
  • Casillas resaltadas dentro del flujo de texto.
  • Áreas de la página con un propósito específico, como espacios publicitarios.
  • Galerías de imágenes

Al agregar atributos de class y / o id a sus elementos div, puede usar CSS para diseñar y posicionar los divs. Esta es la base para crear diseños de página basados ​​en CSS.

A continuación, se muestra un ejemplo que utiliza el elemento div para incluir el contenido de una barra lateral en la página:

    
    <div id="sidebar">
        <h1>Sidebar Heading</h1>
        <p>Sidebar text</p>
        <p>More sidebar text</p>
    </div>

10. <span>...</span>Un contenedor en línea para contenido

El elemento span es similar div en que se usa para agregar estructura a su contenido. La diferencia es que div es un elemento a nivel de bloque, mientras que span un elemento en línea:

Los elementos a nivel de bloque, como div, h1 y p, son elementos que están diseñados para contener bloques de contenido relativamente grandes o independientes, como párrafos de texto. Un elemento a nivel de bloque siempre comienza en una nueva línea.

Los elementos en línea , como span, a e img, están diseñados para contener piezas de contenido más pequeñas, como unas pocas palabras o una oración, dentro de un bloque de contenido más grande. Agregar un elemento en línea no hace que se cree una nueva línea. Los elementos a nivel de bloque pueden contener elementos en línea, pero los elementos en línea no pueden contener elementos a nivel de bloque. Al igual que con a div, a menudo agrega un class y / o id atributo a span para que pueda diseñarlo usando CSS.

El siguiente ejemplo utiliza elementos span para indicar nombres de productos dentro de un párrafo. Estos nombres de productos podrían luego resaltarse usando CSS. Un motor de búsqueda personalizado también podría usar la información proporcionada por los elementos span para identificar los productos dentro de la página. 



    <p>Some of our products include <span class="product">SuperWidgets</span>, <span class="product">MegaWidgets</span>, and <span class="product">WonderWidgets</span>.</p>
